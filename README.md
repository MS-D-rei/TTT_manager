# README

## 開発言語
- Ruby 3.0.1
- Ruby on Rails 6.0.3

## 就業Termの技術
- お気に入り機能
- コメント機能
- devise

## カリキュラム外の技術
- Vue.js

## 実行手順
```
git clone git@github.com:MS-D-rei/TTT_manager.git
cd TTT_manager
bundle install
rails db:create db:migrate
rails db:seed
rails server
```
